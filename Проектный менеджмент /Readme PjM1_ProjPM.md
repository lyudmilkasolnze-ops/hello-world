# Проекты и процессы в управлении

Этот проект — вводный модуль курса по IT-проектному менеджменту.

## Цель проекта:
Научиться различать проекты и процессы, понять базовые понятия менеджмента и попробовать применить их на практике.

## Изучила:
Что такое проект (ограничен по времени, имеет конкретную цель) и процесс (повторяется постоянно);
Основы общего менеджмента: планирование, организация, мотивация, контроль;
Специфику управления именно IT-проектами (техническая экспертиза, работа с требованиями, управление рисками);
Как планировать бюджет, ресурсы, сроки и распределять роли в команде.

## Практические задания:
1. Сравнить проект (разработка промо-сайта с конкурсом) и процесс (его поддержка) по 7 параметрам: цели, длительность, ресурсы, бюджет, результаты, ответственность, контроль.
2. Сравнить IT-проект (мобильное приложение для подсчёта калорий) и не-IT проект (организация stand-up концерта), добавив анализ технологических рисков.

## Итог:
После выполнения вы сможете чётко артикулировать разницу между проектной и процессной деятельностью, оцените свои силы в планировании и заложите базу для позиции Junior Project Manager.
