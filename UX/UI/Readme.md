# Основы UX/UI Дизайна и Интеграция AI

Я освоила FIGMA и внедрила искусственный интеллект в дизайн. Моя работа заключалась в работе с векторной графикой, анализировать интерфейсы и использовать нейросети. Обучилась базовым принципам UX/UI. В проекте показала процесс декомпозиции интерфейсов и применила AI для ускорения прототипа.

## Функционал

  Figma-файл с организованным рабочим пространством.
  Разбор UI-кита популярных приложений.
  Библиотека элементов интерфейса.
  AI-ассеты: иллюстрации, тексты и макеты.

## Выполненная работа

  Создала аккаунт и настроила рабочее пространство в Figma.
  Выбрала популярные приложения и сделала скриншоты их экранов.
  Разобрала интерфейсы на составные элементы и сгруппировала их по типу.
  Сгенерировала иллюстрацию и тексты с помощью нейросетей.
  Разместила все материалы в едином файле для удобства просмотра.

## Инструменты

  Figma — основной инструмент проектирования.
  Midjourney / DALL·E — генерация изображений.
  ChatGPT — написание текстов (UX Writing).
  Uizard / Galileo AI — быстрое прототипирование.
